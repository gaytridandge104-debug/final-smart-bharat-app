# Smart Bharat

AI-powered landing page for government services, civic issue reporting, updates, and citizen assistance.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The app builds into the `dist` folder.

## GitHub Pages Deployment

This project includes `.github/workflows/deploy.yml`, which builds and publishes the app to GitHub Pages when code is pushed to `main` or `master`.

In the GitHub repository, open Settings -> Pages and choose GitHub Actions as the source. After the workflow finishes, use the Pages URL shown by GitHub.
